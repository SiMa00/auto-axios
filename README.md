

# 简介
使用 `ts` 语言对一般常用函数提取封装

# 使用
```
yarn add auto-tools 或 npm i auto-tools 

import ATTools from "auto-tools" // 全量引入
import { isEmpty, isNotEmpty } from "auto-tools" // 按需引入
```
